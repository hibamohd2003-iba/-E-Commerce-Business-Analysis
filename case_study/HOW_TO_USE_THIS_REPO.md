## 👩‍💻 For Data Analysts / Technical Reviewers

**Time needed: 20-30 minutes**

1. Check `sql/` folder — 50+ queries across 5 domains
2. Run queries in SQLite Studio against the Olist dataset
3. Review `analysis/` folder for findings from each section
4. Check `python/analysis.py` for Pandas reproduction of SQL findings

### How to Run the SQL Queries
1. Download dataset from Kaggle
2. Open SQLite Studio and create new database
3. Run sql/01_setup.sql to create all 9 tables
4. Import CSV files into each table
5. Run any of 02 through 06 SQL files

### How to Run the Python Scripts
Run these in your terminal inside the python folder:
- pip install -r requirements.txt
- python analysis.py
- python visualizations.py
- python churn_prediction.py
